# Images Directory

```
public/images/
├── products/      # Product images
├── banners/      # Homepage banners
├── categories/   # Category images
└── placeholders/ # Placeholder images
```

## Usage

Place images in the appropriate folder:
- Product photos: `/images/products/`
- Banner images: `/images/banners/`
- Category thumbnails: `/images/categories/`

## Supported Formats
- JPG, JPEG
- PNG
- WebP
- SVG
